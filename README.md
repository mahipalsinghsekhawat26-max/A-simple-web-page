# A-simple-web-page

This is a beginner-friendly web page I built to practice my coding skills. It contains two simple tools made with basic web technologies.

##  Features
* **Stock Profit/Loss Checker:** Enter your buy and sell prices to quickly see if your trade made a profit or a loss.
* **Percentage Maker:** Enter the marks you obtained and the total marks to calculate your exact percentage.

##  Tech Stack
* **HTML:** For the structure of the page.
* **CSS:** For basic styling and colors.
* **JavaScript:** For the interactive calculators and logic.

##  How to Run
There is no setup required! Just download the code and double-click the `.html` file to open it in any web browser.
